# Electronize
Template to start an Electron project! 

## Start

```
npm start 
```

## Install

```
npm i
cd src
npm i
```

## Build

```
npm run build
```

Check ./dist directory.

## Pack

```
npm run dist
```

And check the ./dist directory.
